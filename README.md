# 🍔 Responsive Food Ordering Website

A modern and fully responsive food ordering web application built using HTML, CSS, and JavaScript. It allows users to browse food items, add them to the cart, and manage their orders with a smooth and interactive UI.

---

## 🚀 Live Demo
👉 https://shubhamshubh1105.github.io/responsive_food_ordering_website/

---

## ✨ Features

- 🛒 Add to Cart functionality
- ➕ Increase / ➖ Decrease item quantity
- ❌ Remove items from cart
- 💰 Dynamic total price calculation
- 📱 Fully responsive (Mobile + Tablet + Desktop)
- 🍔 Dynamic product rendering using JSON
- 🎞️ Swiper slider for reviews section
- 🍽️ Clean and modern UI design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Swiper.js
